Component 6 — Focus Indicator Fix

Issue: ozocoffee.com applies outline: none globally to all a, button, and input elements, removing all keyboard focus visibility.

  • Override with :focus-visible — not :focus — so mouse users are unaffected
  • Dark-background elements (brown nav) get a white outline
  • Light-background elements (form inputs/buttons) get a dark outline

WCAG: 2.4.7 Focus Visible (Level AA)
