Component 5 — Newsletter Email Label Fix

Issue: The email input had no <label>, no aria-label, and no aria-labelledby. Placeholder text alone fails WCAG 1.3.1 (Info and Relationships).

  • Added <label for="newsletter-email"> linked to the input
  • Added id="newsletter-email" to the input
  • Label is visually hidden with the standard clip pattern so it doesn't break the design

WCAG: 1.3.1 Info and Relationships (Level A)
